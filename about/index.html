About

I am Rajat Gupta!

Passionate security researcher working as a Research Scholar at UC, Santa Barbara. I am also GSoC'19 student, worked with The Honeynet Project.

I am also an active Open Source Contributor. Some of my contributions and projects are:

  1. SNARE: Super Next Generation Advanced Reactive Honeypot.
  2. TANNER: Classification and Analysis tool for the Honeypot.
  3. Mitmproxy: An interactive TLS capable intercepting HTTP proxy
  4. Decentalized Blockchain network fetauring a proof-of-work, consensus algorithm and a broadcasting system for the network nodes (private as of now!)
  5. ANTON: An awesome remote administration tool

You can stalk my github for more updated list or feel free to contact me..
